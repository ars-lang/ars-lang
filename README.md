# ArsLanguage

<h3>Welcome to ArsLanguage!</h3>
How to run?<br>
In Unix systems:<br>
<code>./ars main.ars</code><br>
.ars - file extension<br>
main - file name<br>
<br>
In Windows:<br>
Please add file extension exe to file ars!<br>
<code>ars.exe main.ars</code><br>
.ars - file extension<br>
main - file name<br>
<br>
